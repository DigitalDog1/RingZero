## ‚ú® What can RingZero do?

Unlike standard chatbots, RingZero has access to your system through secure tools. You can ask it to:

*   **üìÇ Manage files:** Create, read, edit, and delete files on your PC.
*   **üíª Launch apps & games:** Ask it to open your favorite game or program. If the assistant doesn't know the path, it will prompt you once and remember it forever.
*   **üåê Web search:** Quickly search the internet using DuckDuckGo directly in the chat window, without opening a browser.
*   **üìä Create presentations:** Say "Make a presentation about space", and RingZero will generate a ready-to-use `.pptx` file *(experimental)*.
*   **‚è∞ Set timers and tasks:** "Remind me in 15 minutes to check my email" ‚Äî the agent will track the time and execute the task or notify you.
*   **üëÅÔ∏è Image analysis:** Attach an image (üìé button), and if your chosen model supports Vision, the assistant will describe it or solve problems based on the photo.
*   **üêç Write and execute code:** The assistant can write Python scripts, create small games, and generate instructions on how to run them.

## üöÄ Quick Start Guide

RingZero is portable and ready to use out of the box, but it requires an API connection to an LLM provider to "think".

### Step 1: Installation
1. Go to the [Releases](https://github.com/DigitalDog1/RingZero/releases) section.
2. Download the `RingZero.exe` file.
3. Place it in any convenient folder on your PC and run it (no installation required).

### Step 2: API Setup
Upon first launch, you need to connect a neural network:
1. Click the gear icon (‚öôÔ∏è) in the top right corner of the interface.
2. Enter your **API Key** and **Base URL** from your preferred provider (OpenAI, OpenRouter, Groq, VLLM, etc.).
3. In the "Models" section, click "+ Model" and add the exact names of the models you plan to use (e.g., `gpt-4o`, `qwen-vl-max`).
4. Click "Save all".

### Step 3: Start Chatting
Select the desired model from the top dropdown list, type your request in the bottom field, and click "Execute" (or press Enter).
